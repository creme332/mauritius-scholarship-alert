Mauritius Scholarship Alert

An open-source email notification system for scholarship communiques posted on the Ministry of Education of Mauritius website .

Why?

The scholarship website lacks an email subscription, RSS feed, and API, requiring manual checks for new scholarship updates. This tool eliminates this hassle by automatically sending you an email notification whenever a scholarship is posted. Additionally, it offers advanced features beyond a standard RSS/Atom feed, including customizable filtering options and deadline reminders.

Features

  • Dynamic RSS/Atom feed
  • Customizable email templates
  • Filtering options
  • Deadline reminders

Usage

With Github Actions

This is the recommended approach to use all the features available. The latest instructions on how to use this project are available here.

With RSS/Atom reader

The easiest way to get notified when a new scholarship is posted is to use our RSS/Atom feed. Examples of free RSS/Atom readers are FeedBro and Mozilla Thunderbird. Then, add the following URL to your RSS/Atom reader:

https://creme332.github.io/mauritius-scholarship-alert/feed.xml

However, this method does not support advanced features such as filtering options and deadline reminders.
